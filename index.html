Overview


Whipcream is a simple "classless" css framework that I made as a style boilerplate

for my projects. The idea is to avoid the use of classes as much as possible making it

easier to customize or just use it as is.


Don't expect for a grid system...yet, so for a while you can use dope's lemonade grid system.


Whipcream is still a work-in-progress, you can help me make it better, check it out on GitHub.


Configuration

scss/_config.scss


In this file, all configurable variables are currated especially the $base-em which many

elements' sizes are depending on. I recommend leaving it equal to 10 pixels and change the

$base-font-size's value instead.
